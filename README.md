# DCCLA (PyTorch)

Official PyTorch implementation of DCCLA. 
Our DCCLA ranks first on [JRDB 2019](https://jrdb.erc.monash.edu/leaderboards/detection) and [JRDB 2022](https://jrdb.erc.monash.edu/leaderboards/detection22) 3D pedestrian detection leaderboards.
